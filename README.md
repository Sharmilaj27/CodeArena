# CodeArena — Competitive Coding & AI Coaching Platform

CodeArena is a modern, interactive web application designed to help students and aspiring engineers practice DSA problems, receive AI-style feedback, and track their coding performance — all in one place.

This project simulates a real coding platform (like LeetCode) with a built-in AI coach, dashboard analytics, and a code editor environment.

🌟 Features
🧠 AI Coach (Simulated)
Get instant feedback on code submissions
Error hints and suggestions (without revealing full solution)
Helps improve problem-solving skills
📚 Problem Bank
Curated DSA problems
Filter by:
Difficulty (Easy, Medium, Hard)
Topic (Arrays, DP, Trees, etc.)
Search functionality
💻 Code Editor
Built using CodeMirror
Supports:
Python
JavaScript
C++
Java
Features:
Syntax highlighting
Auto bracket closing
Line numbers
📊 Dashboard
Track:
Problems solved
Accuracy
Attempt history
Weakness analysis by topic
Visual performance chart
🌙 Dark / Light Mode
Toggle between themes
Saved in browser using localStorage
📝 Notes System
Add personal notes per problem
Auto-saved locally
🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
Styling: Tailwind CSS
Editor: CodeMirror
Storage: LocalStorage (for progress & notes)


⚠️ Note
This is a static demo project
AI feedback is simulated using logic (not real AI yet)
No backend or database is used
